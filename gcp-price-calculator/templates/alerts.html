ℹ️ How It Works
  1. Fill in SMTP details → click Validate SMTP to test connection
  2. If validation passes → click Save Configuration (persisted in PostgreSQL)
  3. Choose a Send Mode: Manual (on-demand only), Daily (every day at set time), Weekly (chosen weekdays), Monthly (1st of month)
  4. The scheduler compares After-Discount totals across the last 2 billing runs
  5. Sends email: Previous ₹9,000 → Current ₹9,100 = ▲ ₹100 increased
  6. Use Send Test Now to trigger a one-off send regardless of schedule
